Light Wallet Beta

WARNING: Do NOT install this wallet on an active farming machine. If you do, then it will overwrite your current install and YOU WILL NO LONGER BE FARMING! In case that wasn’t clear enough,

DO NOT INSTALL THIS ON A MACHINE THAT IS ACTIVELY FARMING!

That said, our light wallet uses an electrum-style protocol to reduce syncing from hours to just minutes, or even seconds. It’s also our first wallet implementation supporting Chia Asset Tokens (CATs). We'll make this available to be used with on the same machine as a farmer in a future release.

If you do happen to ignore the above warning and overwrite your farmer, you can still get it back. Data for your farm and full node are not deleted. You can always re-run the Chia farming installer to return to farming, which will delete your light wallet.
